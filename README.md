# Guitar Tech Signature

**Guitar Tech Signature** is an exploratory web application that profiles guitarists based on the **distribution of playing techniques** that define their musical identity.

Rather than ranking guitarists by speed, virtuosity, or genre expectations, this project focuses on *what techniques are used, how often they appear, and what role they play* in shaping a guitaristâ€™s sound.

The result is not a score, but a **signature**.
